Getting Started

Quick Start

MingAI requires no account, no installation, and no setup. Open the link below and start chatting immediately.

‚úÖ

No sign-up required. MingAI runs entirely in your browser. Your chat history is saved to your device's local storage.

1. Open the App

Navigate to the MingAI web app in any modern browser:

URL
https://mingai-nexucore.github.io/

2. Choose a Model

Use the model tabs above the input field. Select MingAI 2.4 for text conversations or MingImage 1.3 to generate images from text descriptions.

3. Type your prompt

Type anything in the input box and press Enter to send. Use Shift+Enter for a new line.

4. Explore Features

Click the ‚äï button next to the input to access additional capabilities like file attachments and online web search.

API Reference

The MingAI API

Work in Progress ‚Äî Not yet publicly available
üöß
API is currently under development

The MingAI public API is a work in progress. The documentation below is a preview of the planned API structure. Endpoints, parameters, and authentication methods are subject to change before the official release. Do not use these endpoints in production applications yet.

The MingAI API will allow developers to programmatically access the MingAI 2.1 language model and MingImage 1.3 image generation model. Once released, you'll be able to integrate MingAI's capabilities directly into your own applications, workflows, and tools.

üì¨

Want to be notified when the API launches? Reach out to the NexuCore team at ming.nexucore@gmail.com to express interest.

Planned Endpoints

The following endpoints are planned for the initial API release:

POST /v1/chat/completions

Send a message and receive a response from MingAI 2.1. Supports both streaming and non-streaming modes.

POST /v1/images/generate

Generate an image from a text prompt using MingImage 1.3.

GET /v1/models

Returns a list of all available MingAI models and their metadata.

Request Format (Preview)

The planned request structure for the /v1/chat/completions endpoint. This is subject to change.

JSON ‚Äì Request Body 
{
  "model": "mingai-2.1",
  "messages": [
    {
      "role": "system",
      "content": "You are a helpful assistant."
    },
    {
      "role": "user",
      "content": "What is the capital of France?"
    }
  ],
  "stream": false,
  "max_tokens": 1024
}
Parameter Type Description
model string The model to use. Currently mingai-2.1.
messages array Array of message objects. Each message has a role (system, user, or assistant) and a content string.
stream boolean If true, the response will be streamed as Server-Sent Events. Defaults to false.
max_tokens integer Maximum number of tokens to generate. Optional.

Authentication (Preview)

The API will use Bearer token authentication. Include your API key in the Authorization header with every request.

HTTP Header 
Authorization: Bearer YOUR_API_KEY
Content-Type: application/json

Error Codes (Preview)

Code Status Description
401 Unauthorized Invalid or missing API key in the Authorization header.

App Features

Chat Interface

MingAI's web app provides a clean, fast interface for interacting with the models. Here's an overview of what's available.

Chat History

All conversations are automatically saved to your browser's local storage. Open the sidebar (the ‚ò∞ menu) to see and resume past conversations. You can also delete individual chats from the sidebar.

System Prompt

You can define custom AI behavior using a system prompt. Open Settings ‚Üí AI Behavior / System Prompt and write instructions for how MingAI should respond. For example: "Always respond in bullet points." or "You are a senior software engineer. Be concise and technical."

Markdown & Code Rendering

All AI responses support full Markdown rendering, including headers, bold/italic text, tables, and syntax-highlighted code blocks. You can copy individual code blocks with the copy button that appears on hover.

Image Generation

Switch to the MingImage 1.3 tab to generate images from text. Type a description in any language ‚Äî MingImage automatically translates your prompt to English for best results before generating.

Tips for Better Images

Be descriptive. Include style, mood, lighting, and subject details. Here are some example prompts:

Generated images can be downloaded using the download button that appears below the image. Right-click saving is intentionally disabled.

File Attachments

Click the ‚äï button in the input area and select Attach File to upload one or more files. MingAI will read the file contents and you can ask questions about them.

Supported File Types

.txt.py.js.ts .jsx / .tsx.html.css .json.yaml / .yml.md .java.c / .cpp / .cs.php .rb.go.rs.sql .xml.sh.lua.r.pdf

File contents are included directly in the message context. For large files, content is automatically truncated at 80,000 characters. You can attach multiple files at once.

Image Vision

MingAI 2.4 can analyze images you attach to your messages. Click the ‚äï button, select Attach File, and choose an image file. MingAI will examine the image and respond to your question about it as if it can see it directly.

Supported Image Types

.png .jpg / .jpeg .gif .webp .bmp

How to Use

Attach an image and ask anything about it ‚Äî "What's in this image?", "Describe this diagram", "What does this chart show?", or "Is there any text in this image?". MingAI will analyze the full visual content and respond in detail.

üëÅÔ∏è

Image analysis happens automatically when you send a message with an attached image. A thumbnail preview is shown in both the input bar and the chat bubble for reference.

PDF Analysis

MingAI 2.4 can read and analyze PDF files you attach to your messages. Click the ‚äï button, select Attach File, and choose a .pdf file. MingAI will extract and interpret the document's contents and respond to any questions you have about it.

How to Use

Attach a PDF and ask anything ‚Äî "Summarize this document", "What are the key points?", "Extract all the dates mentioned", or "Explain section 3 in simple terms". MingAI will analyze the full content and respond in detail.

üìÑ

Attached PDFs are displayed with a red file icon in the chat for easy identification. PDF analysis is powered by MingAI's vision capabilities and works alongside image attachments in the same message.

Text-to-Speech

Every AI response includes a üîä read-aloud button below the message. Click it to have MingAI's response spoken aloud in your browser. Click it again to stop playback at any time.

Automatic Language Detection

TTS automatically detects the language of the AI's response and reads it back in the correct language ‚Äî no configuration needed. Detection is based on script recognition (for Arabic, Japanese, Korean, Chinese, Cyrillic, etc.) and word-frequency scoring for Latin-script languages like French, German, Spanish, Italian, and more.

üîä

Text-to-Speech uses your browser's built-in Web Speech API. Voice availability and quality may vary depending on your browser and operating system. Works best in Chrome and Edge.

Online Web Search

Enable the Online Research toggle (via the ‚äï menu) to give MingAI access to real-time web search results before it responds. This allows MingAI to answer questions about recent events, live data, or anything beyond its training knowledge.

üåê

When online research is active, a Search badge will appear next to the input field. Web search results are automatically appended to your message context before the AI responds.

Web search is powered by Tavily. Results are limited to the top 5 most relevant sources for each query.

Customization

Theme

MingAI supports both a Dark Mode (default) and a Light Mode. Toggle the theme in Settings ‚Üí Light Mode. Your preference is saved locally.

Language

The app interface can be switched to 17 different languages from the Settings panel. Changing the language updates all UI text, placeholder strings, sidebar labels, and suggestion cards.

FAQ

Frequently Asked Questions

Is MingAI free to use?

Yes. The MingAI web app is completely free. No account, no credit card, no sign-up required.

Where is my chat data stored?

All chat history is stored in your browser's local storage. Nothing is uploaded to any server by the app itself. Clearing your browser data will erase your history.

What is MingAI built on?

MingAI is built by NexuCore. The underlying model and infrastructure are proprietary to NexuCore. MingAI will never disclose or reference any third-party AI providers or technologies.

Is MingAI free to use?

Yes. The MingAI web app is completely free. No account, no credit card, no sign-up required.

Can MingAI see and understand images?

Yes. MingAI 2.4 supports image vision. Attach any image to your message and MingAI will analyze its contents and answer questions about it. Supported formats include PNG, JPG, GIF, WEBP, and BMP.

Can MingAI read PDF files?

Yes. MingAI 2.4 supports PDF analysis. Attach any PDF to your message and MingAI will read and interpret its contents. You can ask for summaries, extract specific information, or ask detailed questions about the document.

When will the API be available?

The MingAI API is currently a work in progress. There is no official release date yet. If you're interested in early access or want to be notified at launch, contact ming.nexucore@gmail.com.

Can I use MingAI for commercial projects?

Please reach out to NexuCore directly at ming.nexucore@gmail.com regarding commercial use terms and licensing.

Can I embed MingAI into my website?

Not officially at this time. Once the API is released, you'll be able to integrate MingAI's capabilities into your own apps. Stay tuned for updates.
